Former "American Idol" co-hosts Randy Jackson and Harry Connick Jr. have agreed to host a benefit concert in September. Jackson and Connick Jr. will co-host Louisiana Rising: A Benefit Concert for Flood Relief on Labor Day, Sept. 5. The event will run from 7 p.m. to 9 p.m. in the River Center Theater. Michael Day, general manager of the River Center, confirmed details on Friday morning. Day added that more information would be available next week. The concert will feature performances from more than a dozen artists, including Better Than Ezra, Sonny Landreth, Chris Thomas King, MacKenzie Bourg and Luther Kent. A complete lineup is said to be released in the coming days. Longtime fixture in the Baton Rouge scene, Johnny Palazzotto, will serve as music director of the show. The show will be broadcast and will benefit the American Red Cross's Louisiana Flood Relief fund. Jackson, who was born in Baton Rouge, said he wants to help raise awareness at the event.  "Saddened by the devastating tragedies that keep affecting my Louisiana," Jackson said. "I want to help restore hope to people that are hurting in a place that I will always call home." In a similar statement, Connick Jr. said he was hopeful that the event will bring "much needed attention to the good people of Louisiana." "So sorry that my home state is undergoing such hardship once again," Connick Jr. said. "Seems like just yesterday that Katrina left a path of destruction."

        function populateInfinitySet(sURL) {
            sURL = scrubURL(sURL);

            // Until the origin paramter for cms.search.mlt functions, we have to start/stop the looping on our own
            // This will require not displaying any assets until we hit oAsset.
            // Then we loop through and display until we've been through everything up to our origin

            $.ajax(sURL).then(function(oResponse, sStatus, oXHR) {

                var sNextUrl = null;

                if (typeof oResponse.next_url == 'undefined') {
                    // URL supplied is no good. Revert to original URL
                    sNextUrl = sOriginURL;
                } else {
                    sNextUrl = scrubURL(oResponse.next_url);
                }

                // Manage response
                if (typeof oResponse.assets != 'undefined' && oResponse.assets.length > 0) {
                    
                    // Remove old more data
                    $("#infinite-more").remove();

                    
                    // Build infinity set
                    $.each(oResponse.assets, function(index) {
                            if (index == 1 && sOriginID == null) {
                                sOriginID = oResponse.assets[0].id;
                            }
                        // Display assets once origin is found
                        if (bFoundOrigin == true) {
                            if (this.id == sOriginID) {
                                bInfiniStop = true;
                                sNextUrl = null;
                                $("#infinite-more").remove();
                                
                            } else if (bInfiniStop == false && this.id != '2f908aa0-6ba3-11e6-a586-93921009975d') {
                                // Display asset
                                var sHTML = '';
                                if (this.content) {
                                    if (sInfinityType == 'summary_sticky_siderail') {
                                        sHTML += '<div class="infinite-item original row"><div class="col-md-12">' + this.content + '</div></div><hr>';
                                    } else {
                                        sHTML += '<div class="infinite-item original row"><div class="col-md-8">' + this.content + '</div><div class="col-md-4 text-center">';
                                        if (bShowAd == true) {
                                            // Show advertisement
                                            sHTML += '<div class="infinity-ad empty"></div>'
                                            bShowAd = false;
                                        } else {
                                            bShowAd = true;
                                        }
                                        sHTML += '</div></div><hr>';
                                    }
                                }
                            }
                        } else {
                            if (this.id == sOriginID) {
                                // Origin found. Begin displaying assets
                                bFoundOrigin = true;
                            } else {
                                // Origin found is false. Skip asset
                                // Fail-safe in case origin is not present in set. This can be removed when origin is fixed.
                                if (bFirstRun == true) {
                                    // Stored first id
                                    sFirstID = this.id;
                                    bFirstRun = false;

Former "American Idol" co-hosts Randy Jackson and Harry Connick Jr. have agreed to host a benefit concert in September. Jackson and Connick Jr. will co-host Louisiana Rising: A Benefit Concert for Flood Relief on Labor Day, Sept. 5. The event will run from 7 p.m. to 9 p.m. in the River Center Theater. Michael Day, general manager of the River Center, confirmed details on Friday morning. Day added that more information would be available next week. The concert will feature performances from more than a dozen artists, including Better Than Ezra, Sonny Landreth, Chris Thomas King, MacKenzie Bourg and Luther Kent. A complete lineup is said to be released in the coming days. Longtime fixture in the Baton Rouge scene, Johnny Palazzotto, will serve as music director of the show. The show will be broadcast and will benefit the American Red Cross's Louisiana Flood Relief fund. Jackson, who was born in Baton Rouge, said he wants to help raise awareness at the event.  "Saddened by the devastating tragedies that keep affecting my Louisiana," Jackson said. "I want to help restore hope to people that are hurting in a place that I will always call home." In a similar statement, Connick Jr. said he was hopeful that the event will bring "much needed attention to the good people of Louisiana." "So sorry that my home state is undergoing such hardship once again," Connick Jr. said. "Seems like just yesterday that Katrina left a path of destruction."

        function populateInfinitySet(sURL) {
            sURL = scrubURL(sURL);

            // Until the origin paramter for cms.search.mlt functions, we have to start/stop the looping on our own
            // This will require not displaying any assets until we hit oAsset.
            // Then we loop through and display until we've been through everything up to our origin

            $.ajax(sURL).then(function(oResponse, sStatus, oXHR) {

                var sNextUrl = null;

                if (typeof oResponse.next_url == 'undefined') {
                    // URL supplied is no good. Revert to original URL
                    sNextUrl = sOriginURL;
                } else {
                    sNextUrl = scrubURL(oResponse.next_url);
                }

                // Manage response
                if (typeof oResponse.assets != 'undefined' && oResponse.assets.length > 0) {
                    
                    // Remove old more data
                    $("#infinite-more").remove();

                    
                    // Build infinity set
                    $.each(oResponse.assets, function(index) {
                            if (index == 1 && sOriginID == null) {
                                sOriginID = oResponse.assets[0].id;
                            }
                        // Display assets once origin is found
                        if (bFoundOrigin == true) {
                            if (this.id == sOriginID) {
                                bInfiniStop = true;
                                sNextUrl = null;
                                $("#infinite-more").remove();
                                
                            } else if (bInfiniStop == false && this.id != '2f908aa0-6ba3-11e6-a586-93921009975d') {
                                // Display asset
                                var sHTML = '';
                                if (this.content) {
                                    if (sInfinityType == 'summary_sticky_siderail') {
                                        sHTML += '<div class="infinite-item original row"><div class="col-md-12">' + this.content + '</div></div><hr>';
                                    } else {
                                        sHTML += '<div class="infinite-item original row"><div class="col-md-8">' + this.content + '</div><div class="col-md-4 text-center">';
                                        if (bShowAd == true) {
                                            // Show advertisement
                                            sHTML += '<div class="infinity-ad empty"></div>'
                                            bShowAd = false;
                                        } else {
                                            bShowAd = true;
                                        }
                                        sHTML += '</div></div><hr>';
                                    }
                                }
                            }
                        } else {
                            if (this.id == sOriginID) {
                                // Origin found. Begin displaying assets
                                bFoundOrigin = true;
                            } else {
                                // Origin found is false. Skip asset
                                // Fail-safe in case origin is not present in set. This can be removed when origin is fixed.
                                if (bFirstRun == true) {
                                    // Stored first id
                                    sFirstID = this.id;
                                    bFirstRun = false;
                                } else if (this.id == sFirstID) {
                                    // We've started again. Force origin
                                    sOriginID = sFirstID;
                                    bFoundOrigin = true;
                                } // end Fail-safe
                            }
                        }
                        // Append infinity set to container
                        $("#infinite-container").append(sHTML);
                    });

                    // Include block_id on newly added list items
                    $('#infinite-container .infinite-item.original').each(function() {
                        var sHref = scrubURL($(this).find(".card-headline a").attr("href"));
                        
                        // Add to image and headlines links
                        $(this).removeClass('original')
                            .find(".card-headline a, .image a")
                            .attr("href", sHref + "#tncms-source=infinity-scroll-summary-sticky-siderail-latest");
                    });

                    // Check next URL
                    if (sNextUrl) {
                        // if origin hasn't been found yet and we hit next_url. Trigger the set to pull in again.
                        if (bFoundOrigin == false) {
                            // No origin yet. Call the populate function
                            populateInfinitySet(sNextUrl);
                        } else {
                            // Append sNextUrl to infinity set
                            $("#infinite-container").append('<div id="infinite-more" data-next-url="' + sNextUrl + '"><i class="fas tnt-spinner tnt-pulse"></i></div>');
                        }
                    }
                    
                    // Refresh waypoints
                    Waypoint.refreshAll();
                    if (sInfinityType == 'summary_sticky_siderail') {
                        __tnt.regions.stickySide.determineStick($('#sticky-right-rail'), $('#sticky-right-rail-spacer'), 'siderail');
                    }
                } else {
                    // Empty asset set returned. Go back in for a real set
                    populateInfinitySet(sNextUrl);
                }
            });

        // Infinite waypoint
        var waypoint = new Waypoint({
            element: document.getElementById('infinite-container'),

                if (direction == 'down') {

                }

Former "American Idol" co-hosts Randy Jackson and Harry Connick Jr. have agreed to host a benefit concert in September. Jackson and Connick Jr. will co-host Louisiana Rising: A Benefit Concert for Flood Relief on Labor Day, Sept. 5. The event will run from 7 p.m. to 9 p.m. in the River Center Theater. Michael Day, general manager of the River Center, confirmed details on Friday morning. Day added that more information would be available next week. The concert will feature performances from more than a dozen artists, including Better Than Ezra, Sonny Landreth, Chris Thomas King, MacKenzie Bourg and Luther Kent. A complete lineup is said to be released in the coming days. Longtime fixture in the Baton Rouge scene, Johnny Palazzotto, will serve as music director of the show. The show will be broadcast and will benefit the American Red Cross's Louisiana Flood Relief fund. Jackson, who was born in Baton Rouge, said he wants to help raise awareness at the event.  "Saddened by the devastating tragedies that keep affecting my Louisiana," Jackson said. "I want to help restore hope to people that are hurting in a place that I will always call home." In a similar statement, Connick Jr. said he was hopeful that the event will bring "much needed attention to the good people of Louisiana." "So sorry that my home state is undergoing such hardship once again," Connick Jr. said. "Seems like just yesterday that Katrina left a path of destruction."

        function populateInfinitySet(sURL) {
            sURL = scrubURL(sURL);

            // Until the origin paramter for cms.search.mlt functions, we have to start/stop the looping on our own
            // This will require not displaying any assets until we hit oAsset.
            // Then we loop through and display until we've been through everything up to our origin

            $.ajax(sURL).then(function(oResponse, sStatus, oXHR) {

                var sNextUrl = null;

                if (typeof oResponse.next_url == 'undefined') {
                    // URL supplied is no good. Revert to original URL
                    sNextUrl = sOriginURL;
                } else {
                    sNextUrl = scrubURL(oResponse.next_url);
                }

                // Manage response
                if (typeof oResponse.assets != 'undefined' && oResponse.assets.length > 0) {
                    
                    // Remove old more data
                    $("#infinite-more").remove();

                    
                    // Build infinity set
                    $.each(oResponse.assets, function(index) {
                            if (index == 1 && sOriginID == null) {
                                sOriginID = oResponse.assets[0].id;
                            }
                        // Display assets once origin is found
                        if (bFoundOrigin == true) {
                            if (this.id == sOriginID) {
                                bInfiniStop = true;
                                sNextUrl = null;
                                $("#infinite-more").remove();
                                
                            } else if (bInfiniStop == false && this.id != '2f908aa0-6ba3-11e6-a586-93921009975d') {
                                // Display asset
                                var sHTML = '';
                                if (this.content) {
                                    if (sInfinityType == 'summary_sticky_siderail') {
                                        sHTML += '<div class="infinite-item original row"><div class="col-md-12">' + this.content + '</div></div><hr>';
                                    } else {
                                        sHTML += '<div class="infinite-item original row"><div class="col-md-8">' + this.content + '</div><div class="col-md-4 text-center">';
                                        if (bShowAd == true) {
                                            // Show advertisement
                                            sHTML += '<div class="infinity-ad empty"></div>'
                                            bShowAd = false;
                                        } else {
                                            bShowAd = true;
                                        }
                                        sHTML += '</div></div><hr>';
                                    }
                                }
                            }
                        } else {
                            if (this.id == sOriginID) {
                                // Origin found. Begin displaying assets
                                bFoundOrigin = true;
                            } else {
                                // Origin found is false. Skip asset
                                // Fail-safe in case origin is not present in set. This can be removed when origin is fixed.
                                if (bFirstRun == true) {
                                    // Stored first id
                                    sFirstID = this.id;
                                    bFirstRun = false;
                                } else if (this.id == sFirstID) {
                                    // We've started again. Force origin
                                    sOriginID = sFirstID;
                                    bFoundOrigin = true;
                                } // end Fail-safe
                            }

                        $("#infinite-container").append(sHTML);

Former "American Idol" co-hosts Randy Jackson and Harry Connick Jr. have agreed to host a benefit concert in September. Jackson and Connick Jr. will co-host Louisiana Rising: A Benefit Concert for Flood Relief on Labor Day, Sept. 5. The event will run from 7 p.m. to 9 p.m. in the River Center Theater. Michael Day, general manager of the River Center, confirmed details on Friday morning. Day added that more information would be available next week. The concert will feature performances from more than a dozen artists, including Better Than Ezra, Sonny Landreth, Chris Thomas King, MacKenzie Bourg and Luther Kent. A complete lineup is said to be released in the coming days. Longtime fixture in the Baton Rouge scene, Johnny Palazzotto, will serve as music director of the show. The show will be broadcast and will benefit the American Red Cross's Louisiana Flood Relief fund. Jackson, who was born in Baton Rouge, said he wants to help raise awareness at the event.  "Saddened by the devastating tragedies that keep affecting my Louisiana," Jackson said. "I want to help restore hope to people that are hurting in a place that I will always call home." In a similar statement, Connick Jr. said he was hopeful that the event will bring "much needed attention to the good people of Louisiana." "So sorry that my home state is undergoing such hardship once again," Connick Jr. said. "Seems like just yesterday that Katrina left a path of destruction."
